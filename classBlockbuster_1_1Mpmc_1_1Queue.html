Detailed Description

template<typename T, std::size_t Capacity>
class Blockbuster::Mpmc::Queue< T, Capacity >

A lock-free Multi-Producer Multi-Consumer (MPMC) queue.

This queue supports safe concurrent access from multiple producer and consumer threads without locks.

Template Parameters
TThe type of elements stored in the queue.
CapacityThe maximum number of elements the queue can hold. Must be a power of 2.
