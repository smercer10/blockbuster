Detailed Description

template<typename T, std::size_t Capacity>
class Blockbuster::Spsc::Queue< T, Capacity >

A lock-free Single-Producer Single-Consumer (SPSC) queue.

This queue is designed for efficient communication between a single producer thread and a single consumer thread. It uses atomic operations and careful memory ordering to ensure thread-safety without locks.

Template Parameters
TThe type of elements stored in the queue.
CapacityThe maximum number of elements the queue can hold. Must be a power of 2.
